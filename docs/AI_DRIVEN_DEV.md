# AI Driven Development - Taro

1. Ler a spec correspondente em `specs/`
2. Validar decisões em `docs/ARCHITECTURE.md`
3. Implementar a menor entrega funcional coerente
4. Cobrir regra crítica com teste
5. Rodar `pnpm lint`, `pnpm test` e `pnpm build`
6. Registrar o resultado em `changes/`
7. Não introduzir Firebase ou Supabase em runtime
